# 变更日志

本项目的所有显著变更都将记录在此文件中。

本项目版本号遵循 [语义化版本 2.0.0](https://semver.org/lang/zh-CN/spec/v2.0.0.html) 规范。

---

## [0.3.0] - 2026-06-09

在 v0.3.0 版本中，我们对游戏进行了多项核心重构与优化。重点是将区块的生成与网格化计算移入后台 Web Worker 线程，大幅提升了主线程的帧率表现；重构了地形与生态生成系统，使得地形过渡更加自然；此外还改进了物品与生存机制、完善了设置面板的持久化，并对移动端的操作体验进行了细节优化。

### 🌍 地形与世界生成
- **[重构]** 重构为多维正交的地貌与生态生成系统，实现大陆度单源驱动，解耦了生态与高度计算，并引入坡度检测（陡峭崖壁自动生成裸露岩石）。
- **[重构]** 重构了河流、海岸线与海洋的过渡算法，使用样条曲线进行过渡，引入水体连通解算器以实现自然的物理漫流。
- **[重构]** 调整了地下矿石与矿洞生成，将地下矿石分布重构为独立的 3D 噪声生成阶段；微调了矿洞生成公式使其多为水平隧道并设置高度上限。
- **[优化]** 优化了区块加载算法（采用球形加载并平衡水平与垂直距离），优先渲染玩家所处高度的平面，现在世界仅加载 50% 即可进入。
- **[优化]** 限制了花草、仙人掌等植被的生长底座，现在它们只在合法的方块上生成；防止小水潭生成在河道或河谷附近。
- **[修复]** 修复了异步网格化导致的水体区块边缘不透明残留问题。

### ⚡ 渲染与性能优化
- **[新增]** 新增 Web Workers 异步多线程计算框架，将地形生成和区块网格剖分等 CPU 密集型计算移至后台线程，解决了因网格加载导致的主线程卡顿。
- **[优化]** 将最高视距限制提升至 10 区块，默认渲染半径调整为 4 区块。
- **[新增]** 默认请求使用设备的高性能独立显卡进行渲染。
- **[重构]** 基于贪婪网格（Greedy Meshing）算法重构远景渲染，合并同类材质网格以降低渲染开销。

### 🍎 生存与物品系统
- **[新增]** 引入战利品表（LootTable）机制，将方块与生物的掉落物逻辑解耦，方便后续配置。
- **[新增]** 增加了食用食物的过程时间、咀嚼音效以及咀嚼颗粒特效。
- **[新增]** 背包支持物品合并与溢出，将方块/物品的默认堆叠上限设置为 100。
- **[优化]** 调整饥饿与生命机制：饥饿值为零时生命值会持续扣除直至死亡。
- **[优化]** 调整自动跳跃：仅在玩家向前或斜向前移动时触发。
- **[优化]** 实现了长按屏幕持续攻击/挖掘的交互逻辑。
- **[修复]** 修复了放置方块无声音及掉落物/3D 渲染失效的问题。

### 📱 移动端与交互体验
- **[优化]** 优化了移动端视角控制：修复了移动端小屏模式下鼠标无法拖拽转向、移动时无法攻击/点击快捷栏的 bug。
- **[修复]** 修复了打开设置页面时视角被强制重置拉扯的 bug。
- **[修复]** 修复了自然回血时屏幕边缘错误触发闪红的 bug。
- **[修复]** 修复了加载视距越界覆盖与 LOD 同步导致的主线程卡顿问题。

### 🖥️ 系统设置、UI 与其他
- **[重构]** 重构系统设置模块为声明式注册表架构，修复了视距、视场角等设置没有持久化保存的问题。
- **[优化]** 调试面板（F3）升级，新增了真实 GPU 型号诊断、区块平均加载速度等信息，支持持久化保存面板显示状态。
- **[优化]** 统一弹窗、按钮及控件的样式高度和布局间距。
- **[新增]** 支持在背包中为方块和物品显示多语言悬浮介绍。
- **[优化]** 浏览器默认语言自动与系统/浏览器语言匹配（默认中文）。
- **[其它]** 全局重命名项目为 `cloudcraft` 并更新了 LocalStorage 存储键及测试引用。
- **[其它]** 配置了 Cloudflare 部署与 GitHub Actions 自动发版脚本。

## [0.2.0] - 2026-06-06

在 v0.2.0 版本中，我们对游戏进行了全方位的升级。引入了 3D 子区块（Sub-Chunk）重构，极大提升了渲染与地底裁剪性能。同时全面支持了移动端适配、全新的道具与食物/饥饿度系统、动物生态、雷达小地图以及更丰富的自然植被与矿洞地形。

### 📱 移动端深度适配
- **[新增]** 像素风移动端虚拟控制摇杆（十字键支持按键大小调节）、触屏视角旋转与折叠式快捷菜单。
- **[新增]** 系统物理返回键智能拦截，防止在移动设备上因误触意外退出游戏。
- **[优化]** 抽离了自适应缩放机制，使弹窗与设置面板在不同手机屏幕下自适应缩放，并防止触屏时的意外双击放大。
- **[优化]** 全屏机制优化：移动端进入全屏前会自动锁定为横屏，提升沉浸感（开发调试模式下默认不强制锁定）。

### 🌍 世界生成、渲染与性能优化
- **[重构]** 引入 3D 子区块（Sub-Chunk）网格重构，支持视线外/完全地底区块的智能裁剪与剔除，显著降低渲染开销，为高空和深地底场景提供优异性能。
- **[优化]** 地形与世界生成模块重构为分阶段生成流水线，大幅增强了地形、植被与建筑的扩展能力。
- **[优化]** 生产环境打包优化，引入了 Three.js 异步懒加载及代码按需分包，网页首屏加载与首渲染速度显著提升。
- **[优化]** 3D 噪波矿洞生成算法，生成的矿洞走向更曲折、宽度更自然逼真，并支持生成山地露天矿洞入口。
- **[优化]** 自然植被与生态系统：新增多种野外花草植被，搭配精致的新版像素贴图；实现落叶、树苗无碰撞体积及种植生长机制。
- **[优化]** 渲染细节增强：支持透明方块（如水体）内部网格渲染及树叶双面交叉斜面模型，使植被渲染效果更为饱满。
- **[修复]** 修复了地表植被随机浮空、小地图在 3D 区块下无法正常渲染地形与植被颜色的 Bug。

### 🍎 道具、食物与动物系统
- **[重构]** 道具与物品系统底层解耦，设计了独立的道具基类、类型枚举与注册表，支持食物、非方块工具等更丰富的道具扩展。
- **[新增]** 玩家饥饿度系统与饮食机制，支持食物食用（长按或右键食用），割草现在有概率掉落可食用的植物种子。
- **[新增]** 动物生态系统，首个实体“猪”及其 AI 寻路、朝向与渲染上线，击杀可掉落“生猪肉”道具。
- **[修复]** 解决模块循环加载导致的掉落物纹理丢失和树苗渲染错误。

### 🏃 物理与控制优化
- **[重构]** 重写水下物理引擎，采用基于阻尼的游泳推进模型取代生硬的自动浮力，带来自然的游泳姿态、水面起伏和顺畅的登岸体验。
- **[优化]** 玩家第一人称视角高度微调，对齐经典沙盒游戏设定。
- **[优化]** 新增 Shift 键（移动端双击前进键）疾跑加速，以及加速状态下的自动跳跃。
- **[修复]** 修复物理掉落物被非实体方块（如花草）阻挡而在空中悬浮的 Bug。

### 💾 存档与数据管理
- **[新增]** 存档数据中新增记录游戏版本号，为多版本兼容与未来数据迁移提供底座。
- **[优化]** 存档导入流程，导入完毕后世界直接自动加载，缩短进游戏路径。

### 🖥️ UI界面、国际化与设置
- **[新增]** 界面左上角雷达小地图，支持玩家朝向、出生点和周围动物实体的动态指示。
- **[优化]** 重构世界生成加载屏，升级为经典的拟物化异步进度条，取代旧的调试网格显示。
- **[优化]** 完善中英文双语国际化（i18n）文案对齐，并在设置页面附带当前游戏版本号。

## [0.1.0] - 2026-06-04

这是 CloudCraft 的首个版本，包含游戏的基础核心引擎与底座功能。

### 🌍 世界生成与生态
- **[新增]** 支持平原、森林（橡树与白桦树）、针叶林（云杉树）、沙漠（仙人掌与沙丘）、丛林（丛林巨树）、高原以及石峰等多种地形群系。
- **[新增]** 河流与陆地水潭的生成算法，附带智能边缘拉高防溢处理。
- **[新增]** 基于 3D 噪波的地下刻蚀矿洞系统。
- **[优化]** 基于纹理图册材质合批与区块网格合并，极大降低 Draw Call，支持方块修改后网格动态实时重建。

### 📦 游戏机制与交互
- **[新增]** 54 格背包存储和整理界面。
- **[新增]** 物理落沙模拟，沙子受重力影响在失去支撑时会自动下落。

### 🏃 物理与控制
- **[新增]** 自研体素碰撞物理系统，支持玩家行走、游泳、飞行以及精确的 AABB 方块碰撞判定。
- **[新增]** 行走撞墙自动跳跃和半阶高度的自动上台阶检测。

### 💾 游戏存读档
- **[新增]** 基于本地数据库的高效多存档管理，可记录修改的方块、玩家位置以及背包物品。

### 🔊 音效与特效
- **[新增]** 使用 Web Audio 实时合成行走、破坏、放置等基础音效，不依赖外部大型音频文件。
- **[新增]** 方块被破坏时带有对应材质颜色的碎屑散落粒子效果。

### 🖥️ 界面与设置
- **[新增]** 磨砂玻璃质感的像素风 UI 主题，包含开始菜单、暂停菜单、HUD 快捷栏，以及可调节视野与渲染距离的设置面板。
