# WebCraft 3D

WebCraft 是一个运行在浏览器中的轻量级 3D 像素沙盒游戏项目，基于 React 19、Three.js、TypeScript、Zustand 和 Sass 模块化构建。

本项目实现了类似《我的世界 (Minecraft)》的核心基础玩法，并结合了前端的现代工程化实践与渲染性能优化。

---

## 🎮 核心特性

*   **多群系 3D 地形生成**：
    *   基于自定义 3D 噪波（Simplex Noise）算法，支持随机种子生成起伏山峦与平原地形。
    *   支持平原、森林（橡树与白桦树）、针叶林（云杉树）、沙漠（仙人掌与沙丘）、丛林（丛林巨树）、高原以及石峰等多种生态群系。
    *   支持河流与陆地水体生态，具备防溢堤防的智能边缘拉高处理。
    *   地下矿洞系统：基于 3D 噪波刻蚀地下矿洞，并具备防穿透水体保护，支持山地露天入口。
*   **实体与生态系统**：
    *   引入基础动物实体系统，目前支持猪实体及掉落的生猪肉道具。
*   **物理引擎与碰撞检测**：
    *   自研轻量级体素物理系统，支持玩家行走、游泳浮力判定、飞行以及精确的 AABB（轴对齐包围盒）碰撞检测，实现顺畅的墙体阻挡和滑动效果。
    *   支持特定方块物理机制（如沙子受重力影响，在失去下方支撑时会自动下落）。
    *   智能移动辅助：包含行走撞墙自动跳跃、半阶高度的自动上台阶检测。
*   **游戏机制与存读档**：
    *   提供 54 格大容量背包存储和整理界面。
    *   支持多存档管理系统，基于浏览器 IndexedDB API，实时记录修改的方块、玩家位置以及背包物品。
*   **渲染性能与首屏加载优化**：
    *   **纹理图册（Texture Atlas）合批**：将所有方块材质拼合在一张纹理图上，减少 GPU 渲染开销。
    *   **区块网格合并（Chunk Mesh Merge）**：按区块合并网格面，极大地降低 Draw Calls，支持方块破坏与放置时动态重建网格。
    *   **Three.js 异步懒加载**：结合空闲时间预加载技术，大幅减小首屏打包体积，显著优化首屏加载性能。
    *   **Vite 生产分包**：优化生产环境打包策略进行代码分包，消除单 Chunk 过大的编译警告。
*   **交互反馈与合成音效**：
    *   **动态粒子效果**：破坏方块时产生对应方块材质的散落粒子。
    *   **Web Audio 音效系统**：内置轻量音效合成器，实时合成行走、放置、破坏音效，无需依赖外部大体积音频文件。
*   **现代 React UI 交互与国际化（i18n）**：
    *   采用毛玻璃质感与像素风相融合的现代化 UI，包含开始菜单、暂停菜单、HUD 快捷栏、红幕伤害反馈以及设置面板（支持调节视场角 FOV、渲染距离等）。
    *   新增“我的世界”经典风格的世界生成进度弹窗，支持区块异步加载展示。
    *   支持多语言国际化（i18n）翻译，彻底重构并对齐中英文硬编码文案。
    *   在设置弹窗左下角直观展示当前游戏版本信息。

## 🛠️ 技术栈

*   **核心引擎**：Three.js (WebGL 渲染)
*   **界面与视图**：React 19 & Sass / Scss Modules
*   **全局状态管理**：Zustand
*   **国际化**：支持中英文双语切换的 i18n 系统
*   **数据库**：IndexedDB（多存档数据持久化）
*   **工程构建**：Vite & TypeScript
*   **规范约束**：Husky, Lint-staged, Commitlint
*   **单元测试**：Vitest & Testing Library

## 🚀 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```
启动后在浏览器中打开命令行输出的本地服务地址（默认 `http://localhost:5173`）即可开始游戏。

### 3. 项目构建
编译 TypeScript 并打包生产环境静态资源：
```bash
npm run build
```

### 4. 运行单元测试
运行物理、噪波算法等核心计算逻辑的单元测试：
```bash
npm run test      # 交互式测试
npm run test:run  # 单次运行测试
```
